1. ¿De dónde saco la API Key?
Para que ese código funcione, necesitas una cuenta en la plataforma de desarrolladores de OpenAI:

Ve a platform.openai.com.

Regístrate o inicia sesión.

Busca en el menú lateral el icono de una llave o la sección "API Keys".

Haz clic en "Create new secret key".

Cópiala de inmediato y guárdala en un bloc de notas temporal, porque una vez que cierres la ventana, no podrás volver a verla completa.

Nota importante: OpenAI no es gratis para la API. Normalmente te regalan unos dólares de crédito al inicio, pero si no, tendrás que cargar al menos $5 USD para que el código no te dé un error de "Quota exceeded".
